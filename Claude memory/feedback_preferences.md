Toujours proposer un backup avant de modifier un fichier existant important.
**Why:** L'utilisateur l'a explicitement demandé — il veut pouvoir revenir en arrière.
**How to apply:** Avant toute modification d'un fichier de code existant, proposer ou créer une copie .bak ou mentionner la possibilité.

Ne pas exécuter les scripts à sa place — les lui fournir pour qu'il les lance lui-même.
**Why:** Il préfère garder le contrôle sur ce qui s'exécute sur sa machine.
**How to apply:** Fournir les commandes à copier-coller plutôt que de les exécuter via Bash.

Communiquer entièrement en français.
**Why:** Préférence explicite, utilisateur québécois.
**How to apply:** Toujours répondre en français, commentaires de code en français aussi sauf si convention contraire dans le projet.
